## webpack
--> 首先知心 npm init -y 初始化
--> 局部安装 webpack webpack-cli -D 防止影响其他项目

--> package.json 里面配置 "build": "webpack" 编译脚本后，执行npm run build 进行打包

-->创建webpack.config.js配置文件
